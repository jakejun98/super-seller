# 🚨 재고 부족 알림 봇 설계서

## 프로젝트 개요
상품의 재고 수량을 모니터링하여, 설정된 임계값(Threshold) 미만으로 떨어진 상품을 감지하고 관리자에게 경고 메시지(로그)를 출력하는 자동화 스크립트.

---

## 🤖 Phase 2-3: 재고 알림 스크립트 요구사항

### 1. 목표
1.  Supabase `products` 테이블의 모든 데이터를 조회한다.
2.  각 상품의 `stock`(재고) 수치를 확인한다.
3.  재고가 **10개 미만**인 상품을 발견하면, 터미널에 🚨 경고 메시지를 출력한다.
    * 출력 형식: `[경고] '{상품명}' 재고가 부족합니다! (현재: {n}개)`
4.  재고가 10개 이상이면 "✅ '{상품명}' 재고 충분 (현재: {n}개)"를 출력한다.

### 2. 기술 요구 사항
- **언어:** Python 3.x
- **라이브러리:** `supabase`, `python-dotenv`
- **대상 테이블:** `public.products` (컬럼: `name`, `stock`)

### 3. Gemini에게 요청할 프롬프트
> "위 설계서를 바탕으로 `scripts/check_inventory.py` 파이썬 코드를 작성해 줘.
>
> **핵심 지시사항:**
> 1. `.env.local`에서 환경 변수(`NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY`)를 로드해.
> 2. `products` 테이블의 `name`과 `stock` 컬럼을 조회해.
> 3. 반복문을 돌면서 `stock`이 10 미만인 경우와 그렇지 않은 경우를 나눠서 출력해줘.
> 4. 주석은 한글로 달고, 마크다운 기호 없이 순수 코드만 줘."
