# 🤖 AI 고객 문의 자동 분류기 설계서

## 프로젝트 개요
고객의 문의 내용(텍스트)을 분석하여, 사전에 정의된 카테고리(배송, 환불, 상품문의, 기타) 중 하나로 자동 분류하는 AI 스크립트.

---

## 🤖 Phase 2-4: CS 분류기 요구사항

### 1. 목표
1.  미리 준비된 '테스트 문의 데이터' 리스트를 순회한다.
    * 예: "배송이 언제 출발하나요?", "사이즈가 안 맞아요 환불해주세요", "이거 재입고 언제 돼요?"
2.  Google Gemini API(`google.generativeai`)를 호출한다.
3.  AI에게 "이 문장이 [배송, 환불, 상품문의, 기타] 중 어디에 속하는지 단어 하나만 대답해"라고 시킨다.
4.  AI의 답변(카테고리)을 받아서 원본 문의 내용과 함께 출력한다.

### 2. 기술 요구 사항
- **언어:** Python 3.x
- **라이브러리:** `google-generativeai`, `python-dotenv`
- **환경 변수:** `.env.local`의 `GEMINI_API_KEY` 사용

### 3. Gemini에게 요청할 프롬프트
> "위 설계서를 바탕으로 `scripts/classify_cs.py` 파이썬 코드를 작성해 줘.
>
> **핵심 지시사항:**
> 1. `google.generativeai` 라이브러리를 사용해서 Gemini Pro 모델을 설정해.
> 2. API 키는 `os.environ.get('GEMINI_API_KEY')`로 가져와.
> 3. `test_inquiries` 리스트에 한국어 문의 예시 5개를 넣어줘 (배송, 환불, 상품 등 섞어서).
> 4. 반복문으로 각 문의를 AI에게 보내고, 분류 결과만 딱 받아내서 출력해.
> 5. 프롬프트 메시지는 '다음 문장을 [배송, 환불, 상품문의, 기타] 중 하나로 분류해서 단답형으로 대답해: {문의내용}' 형식으로 작성해."
